# movie-web-info-directory
该仓库主要用于建立影视相关网站的基础信息目录，对公开网络中的电影、电视剧及综合影视站点进行系统化整理。内容仅限站点层级的公开信息，不包含任何影视内容或播放链接。所有资料整理自公开渠道，目的在于形成清晰、可查询的站点信息结构，便于后续核对与对照。本项目不参与站点运营，不对任何网站作主观评价，仅作为信息归档与资料整理用途存在。
